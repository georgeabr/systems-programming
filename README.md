# systems-programming
Systems programming

### w32time
It gets time from an NTP server and optionally syncs it
```
$ w32time.exe pool.ntp.org -set
Server pool.ntp.org UTC time: 2025-11-08 12:26:37
System time updated.

$ w32time.exe pool.ntp.org
Server pool.ntp.org UTC time: 2025-11-08 12:26:37

```
